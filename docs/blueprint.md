# **App Name**: HashDrop

## Core Features:

- File Upload Zone: Implement a drag and drop zone for users to upload local files.
- Hash Calculation: Calculate the SHA-256 hash of the uploaded file.
- Hash Display: Display the calculated SHA-256 hash value.

## Style Guidelines:

- Primary color: Neutral gray (#F0F0F0) for background.
- Secondary color: Dark gray (#333) for text.
- Accent: Blue (#4287f5) to highlight the drag and drop zone and buttons.
- Clean and simple layout with a clear drag and drop area.
- Use a file upload icon in the drag and drop zone.
- Subtle animation when a file is dragged over the drop zone.
